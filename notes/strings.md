# Strings

## Key Concepts
- Strings are sequences of characters.
- In C++, `std::string` supports many built-in operations.
- Case handling and ASCII conversions are common in interviews.

## Time Complexity
- Index access: **O(1)**
- Concatenation: usually **O(n)**
- Substring extraction: **O(k)** where k is substring length

## Common Patterns
- Two pointers for palindrome checks
- Frequency counting
- Sliding window for substring problems
- Pattern matching basics
