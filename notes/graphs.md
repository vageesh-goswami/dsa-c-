# Graphs

## Key Concepts
- A set of vertices (nodes) and edges.
- Can be directed/undirected, weighted/unweighted.
- Represented with adjacency list or matrix.

## Time Complexity
- BFS/DFS (adjacency list): **O(V + E)**
- Dijkstra (priority queue): **O((V + E) log V)**

## Common Patterns
- BFS for shortest path in unweighted graph
- DFS for components and cycle checks
- Topological sort (DAG)
- Union-Find for connectivity
