# Linked List

## Key Concepts
- Nodes connected by pointers.
- Dynamic size, easy insert/delete once node position is known.
- No direct random access like arrays.

## Time Complexity
- Access by index: **O(n)**
- Insert/Delete at head: **O(1)**
- Search: **O(n)**

## Common Patterns
- Slow/fast pointers
- In-place reversal
- Cycle detection (Floyd's algorithm)
- Dummy node technique
