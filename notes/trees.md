# Trees

## Key Concepts
- Hierarchical structure with root and child nodes.
- Binary tree node has at most two children.
- BST has ordering property: left < root < right.

## Time Complexity
- Traversals (DFS/BFS): **O(n)**
- Search/Insert in balanced BST: **O(log n)**
- Search/Insert in skewed BST: **O(n)**

## Common Patterns
- DFS (preorder, inorder, postorder)
- BFS (level order)
- Recursion with subtree answers
- Height/diameter style problems
