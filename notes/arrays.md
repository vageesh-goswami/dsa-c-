# Arrays

## Key Concepts
- Contiguous memory structure.
- O(1) random access by index.
- Insert/delete in middle is expensive (shifting elements).

## Time Complexity
- Access: **O(1)**
- Search: **O(n)**
- Insert/Delete at end (dynamic array): **amortized O(1)**
- Insert/Delete at middle: **O(n)**

## Common Patterns
- Prefix sums
- Sliding window
- Two pointers
- Hashing with arrays/maps
