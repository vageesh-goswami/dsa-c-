# DSA C++ Practice Repository

A beginner-friendly Data Structures & Algorithms (DSA) practice repository in C++.

## Folder Structure

- `arrays/` - array basics and interview-style practice problems
- `strings/` - string manipulation and common patterns
- `linked_list/` - linked list implementation and classic problems
- `trees/` - binary tree traversal and utility problems
- `graphs/` - graph representation and traversal fundamentals
- `dp/` - dynamic programming fundamentals
- `sorting/` - sorting algorithms and related practice
- `notes/` - concise topic-wise study notes

## Compile and Run (g++)

From the repository root:

```bash
# Compile

g++ -std=c++17 arrays/arrays_basics.cpp -o /tmp/arrays

# Run
/tmp/arrays
```

Use the same pattern for any file in `arrays/`, `strings/`, `linked_list/`, `trees/`, `graphs/`, `dp/`, or `sorting/`.

## Suggested DSA Interview Study Roadmap

1. **Week 1-2: Arrays + Strings**  
   Practice traversal, prefix sums, sliding window, two pointers.
2. **Week 3: Linked List + Sorting**  
   Learn pointer manipulation, merge sort logic, and partitioning ideas.
3. **Week 4: Trees + Graphs**  
   Master DFS/BFS, recursion, tree properties, and shortest-path basics.
4. **Week 5-6: Dynamic Programming**  
   Start with 1D DP, then 2D DP, then state transitions.
5. **Ongoing**  
   Re-solve old problems, track mistakes, and time yourself in mock interviews.

Happy learning and consistent practice!
